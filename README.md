# Quantum Energy Tracker

Real-time electrical network monitoring dashboard built with React + Lucide icons.

## Features
- **Live Data**: 6 nodes (2 generators, 3 consumers, 1 substation) 
- **700ms refresh** with sine wave simulation + noise
- **Status alerts**: Normal/Green, Warning/Orange, Critical/Red
- **Sparklines** + barcharts + KPIs
- **Node detail view** with stats (current/peak/min/avg)

## Tech
- React 18 + hooks (useState, useEffect, useRef)
- Lucide React icons
- Custom SVG sparkline + barchart components
